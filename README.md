## 🔧 Common Problems

### The app will not open

- Make sure you extracted the ZIP file first
- Try right-clicking the app and choosing Run as administrator
- Check that Windows did not block the file
- Download the latest release again

### The game and trainer do not work together

- Start the game before the trainer
- Keep both running
- Make sure you use the right version of the trainer
- Close the trainer and open it again

### Windows shows a security prompt

- Choose the option to allow the app
- If the file came from the release page, this is normal for many downloaded apps
